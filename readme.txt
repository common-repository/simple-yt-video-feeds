Add a dynamic YouTube video player directly onto your website by downloading Simple Yt Video Feeds plugin.

== Description ==

The Simple Yt Video Feeds can easily be added to your site with all of the Below benefits.

(i) Easy to use.

(ii) Easy to install and download.

(iii) Free support available from wordpress when it is needed.

(iv) Show/Hide Frontend view using setting.

(v) Use Shortcode

Easy to use Different pages or posts or widget using shortcode "[syvf_youtube_feeds_sh]".

Use This Code in Page Template <code> <?php echo do_shortcode('[syvf_youtube_feeds_sh]'); ?> </code>.


== Installation ==

This section describes how to install the plugin and get it working.

1. Install the wordpress plugin first
You can install it via Wordpress Installer or can download the file first then Upload it on /wp-content/plugins/directory

2. Activate the plugin

3. You will see "Simple Yt Video Feeds" settings from right side of your wordpress menu.

Configuration Options:

Youtube Video ID : Put your Youtube video ID

Youtube Video URL : Put the video URL https://www.youtube.com/watch?v=qUGnu0gXtn4


qUGnu0gXtn4 This part only on the field you have to put only

Margin Top - Height & Width : Their you have to Put Numeric Number as per your need.
